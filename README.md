# KCG-CGPA-CALCULATOR-
# KCG Pulse ⚡

### Track Your Academic Heartbeat

KCG Pulse is a highly personalized academic command center designed specifically for KCG College of Technology students.

Unlike a traditional GPA calculator, KCG Pulse acts as a personal academic dashboard that helps students monitor performance, predict outcomes, visualize progress, and stay motivated throughout their engineering journey.

> **Project Note**
>
> The complete concept, feature planning, workflow design, grading logic, user experience, and product vision were created by me.
>
> The application itself was developed using modern AI-assisted development ("vibe coding"), where AI helped accelerate implementation while the core idea, architecture decisions, and product direction remained mine.

# 🚀 Core Features

## 1. Smart GPA & CGPA Engine

### Tailored for KCG Students

Pre-configured with Semester 1 and Semester 2 subjects along with official credit weightages.

### Accurate Grade Calculations

Supports:

* O Grade
* A+ Grade
* A Grade
* B+ Grade
* B Grade
* C Grade
* U Grade

Instantly calculates:

* SGPA
* CGPA
* Credit Points
* Academic Progress

### Real-Time Results

Update grades and instantly see GPA changes without refreshing the page.

## 2. Predictive Analytics & Forecasting

### Hard Work Score™

A custom metric that measures academic effort by comparing obtained marks against maximum possible marks.

### GPA Prediction Engine

Uses:

* CA 1
* CA 2
* Model Examination

to generate:

* Conservative GPA Prediction
* Expected GPA Prediction
* Best Case GPA Prediction

### Probability Forecasting

Shows estimated probability of achieving:

* 7.0+ GPA
* 8.0+ GPA
* 9.0+ GPA

based on current academic performance.

# 🎯 Project Vision

Most students only think about their GPA after results are published.

KCG Pulse aims to change that.

The goal is to help students:

* Track progress continuously
* Identify weaknesses early
* Stay motivated
* Make data-driven academic decisions
* Improve performance before final exams arrive

KCG Pulse transforms academic performance from a static result into a measurable journey.
